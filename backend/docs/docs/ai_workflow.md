# AI Workflow

1. Get live football data.
2. Detect trending football topics.
3. Generate 10 content ideas.
4. Rank the ideas by engagement potential.
5. Generate:
   - Headline
   - Caption
   - Video script
   - Hashtags
6. Send content for human approval.
7. Publish after approval.
8. Track:
   - Views
   - Likes
   - Shares
   - Comments
   - Followers gained
9. Learn which content performs best and improve future suggestions.
